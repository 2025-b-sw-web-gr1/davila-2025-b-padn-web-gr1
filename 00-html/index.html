Introducción

HTML (HyperText Markup Language) es el lenguaje de marcado que estructura el contenido en la web. Aquí verás cómo crear páginas, usar etiquetas, atributos y combinar HTML con CSS y JavaScript.

Estructura básica de un documento

Un documento mínimo tiene la declaración doctype, html, head y body. Dentro de head van meta, title y enlaces a estilos.

<!doctype html>
<html lang="es">
  <head>
    <meta charset="utf-8">
    <title>Mi página</title>
  </head>
  <body>
    <h1>Hola</h1>
  </body>
</html>

Texto y formato

Etiquetas para encabezados, párrafos, énfasis, citas y comentarios.

<h1>Título principal</h1>
<p>Un párrafo con <strong>texto fuerte</strong> y <em>énfasis</em>.</p>
<!-- Comentario en HTML -->

Enlaces (a)

Enlaces a otras páginas, anclas y uso de target para abrir en nueva pestaña.

<a href="https://example.com" target="_blank" rel="noopener">Ir a example</a>
<a href="#seccion">Ir a sección interna</a>

Imágenes

Usa alt para accesibilidad y controla tamaño con atributos o CSS.

<img src="https://via.placeholder.com/180" alt="Ejemplo" width="180">

Listas

Listas ordenadas, desordenadas y de definición.

<ul>
  <li>Elemento A</li>
  <li>Elemento B</li>
</ul>

<ol>
  <li>Paso 1</li>
</ol>

Tablas

Tablas para datos tabulares: thead, tbody, tr, th, td.

<table border="1">
  <thead><tr><th>Nombre</th><th>Edad</th></tr></thead>
  <tbody><tr><td>Ana</td><td>23</td></tr></tbody>
</table>

Formularios e inputs

Elementos básicos: form, input, textarea, select, labels y validación nativa.

<form onsubmit="event.preventDefault(); alert('Enviado');">
  <label>Nombre: <input name="name" required /></label>
  <label>Color: <input type="color" /></label>
  <button type="submit">Enviar</button>
</form>

Multimedia: audio y video

Etiquetas audio y video con controles, subtítulos y formatos compatibles.

<audio controls>
  <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg">
  Tu navegador no soporta audio.
</audio>

<video width="320" controls>
  <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
  Video no disponible.
</video>

Etiquetas semánticas

Usa header, nav, main, article, section, aside y footer para mejorar significado y accesibilidad.

<header><h1>Mi sitio</h1></header>
<main><article><h2>Noticia</h2></article></main>
<footer>Pie de página</footer>

Canvas y SVG

Canvas para dibujo dinámico y SVG para gráficos vectoriales. Aquí un ejemplo sencillo de canvas.

<canvas id="mycanvas" width="280" height="120" style="border:1px solid #333"></canvas>
<script>
  const c = document.getElementById('mycanvas').getContext('2d');
  c.fillStyle = '#6EE7B7'; c.fillRect(10,10,100,60);
  c.fillStyle = '#60A5FA'; c.font='16px sans-serif'; c.fillText('Canvas',20,45);
</script>

Pequeños ejemplos de LocalStorage y Geolocation.

// LocalStorage

// Geolocation (si el navegador lo permite)
navigator.geolocation && navigator.geolocation.getCurrentPosition(pos => console.log(pos.coords));

Accesibilidad y buenas prácticas

  • Usa atributos alt en imágenes y labels en formularios.
  • Usa roles ARIA solo cuando sea necesario.
  • Comprueba el contraste de colores y navegación con teclado.

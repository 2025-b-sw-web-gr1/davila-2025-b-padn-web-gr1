Construyendo con HTML5 semántico

Publicado el 23 de noviembre de 2025 por Equipo

Introducción

Este ejemplo muestra el uso de elementos semánticos de HTML5 para estructurar contenido de forma clara y accesible.

Características principales

  • Uso de <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>.
  • Uso de <figure> y <figcaption> para contenido multimedia.
  • Enlaces skip y roles/atributos ARIA mínimos para mejorar la navegación por teclado/lectores de pantalla.

Etiquetas relacionadas: HTML5, Accesibilidad
